# clojure-yasunori

## Syntax

```
yasunori
yasunori -list
yasunori id [id...]
```

### Non arguments

Show ramdom

- API: `https://api.yasunori.dev/awesome/random`

### `-list`

Show yasunori entries.

- API: `https://api.yasunori.dev/awesome`

### id

Show specified entry of ID number.

- API: `https://api.yasunori.dev/awesome/{id}`

## Build

```bash
make build
```

## Build (native)

```bash
make build.native
```
